# android_device_samsung_sm7450_m55xq
Unmodified stock kernel source for Samsung m55xq based on One UI 8


# Kernel base
Samsung GKI 5.10.226-android12

## Requirements
1. A 64-bit Linux PC (Debian or Ubuntu recommended)
2. Minimum 24 GB of RAM (16 GB RAM with 8-16 GB swap works too) (vmlinux linking kills the build on low RAM devices)
3. Atleast 4 cores of processor
4. Big enough storage but depends


## To compile:

1. Clone the git repo
2. in the source, execute: *sh build_kernel_GKI.sh*
